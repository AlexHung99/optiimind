SQLite Browser & Runtime API for Unity

OptiLiteDB

Unlock the ultimate SQLite workflow inside Unity. Zero DLLs, native performance, and editor-first database tooling.

在 Unity 中解放終極 SQLite 工作流。零 DLL 封裝、原生效能，並提供以編輯器為核心的資料庫工具。

OptiLiteDB combines a feature-rich in-editor DB browser with a transaction-safe C# Runtime API. It binds directly to OS-native SQLite libraries, avoiding cross-platform binary friction while keeping builds lean.

OptiLiteDB 將完整的編輯器視覺化 DB Browser 與交易安全的 C# Runtime API 融合，直接連結系統內建 SQLite 原生庫，降低跨平台二進位配置摩擦，也讓發布包保持輕量。

Key Product Highlights

核心產品亮點

Everything needed to browse, design, import, export, query, and automate local SQLite data without leaving Unity.

從瀏覽、設計、匯入匯出、查詢到自動化，完整處理 Unity 專案中的本機 SQLite 資料。

01

Zero-DLL Architecture

零 DLL 輕量架構

Ships no native binaries. OptiLiteDB dynamically links to OS-native SQLite engines to avoid build conflicts and keep final players small.

免隨包打包任何原生庫，直接連結系統 SQLite 引擎，降低跨平台編譯衝突並縮小發布包體。

02

In-Editor DB Browser

內嵌資料庫瀏覽器

Open database files under Assets, browse with pagination, edit cells directly, sort headers, and set NULL through context actions.

支援 Assets 內資料庫雙擊開啟、分頁瀏覽、直接編輯 Cell、表頭排序與右鍵設為 NULL。

03

Visual Table Designer

視覺化表結構設計

Reorder, rename, add, or drop columns. Schema changes run through transaction-backed rebuilds that restore indexes and triggers.

可自由調整欄位、型別、主鍵與排序，並以交易保護重建資料表，自動還原既有索引與觸發器。

04

Transaction-Safe CSV

交易保護 CSV 匯入匯出

Map CSV columns visually, choose delimiters, and use clear-before-import safely. Failed imports roll back without damaging existing data.

可視覺化對應 CSV 欄位並自訂分隔符，清空匯入模式也在單一交易中執行，失敗時安全 rollback。

05

AI Agent Ready

AI 智慧代理接口

OptiLiteEditorAPI lets AI assistants and editor tooling query or write tables through simple C# one-liners and refresh the UI instantly.

內建 OptiLiteEditorAPI，讓 AI 助理或編輯器工具用簡短 C# 指令讀寫資料表並即時刷新 UI。

06

Production Runtime API

專業級 Runtime API

Parameterized bindings guard against SQL injection, while BEGIN/COMMIT/ROLLBACK blocks handle high-speed save workloads.

支援參數化綁定以防止 SQL 注入，也支援交易區塊處理高速存檔與大量資料寫入。
